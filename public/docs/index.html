Introduction

App to sell ,rent , or buy colthes

Base URL: http://localhost

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

To authenticate requests, include an Authorization header with the value "Bearer {YOUR_AUTH_KEY}".

All authenticated endpoints are marked with a requires authentication badge in the documentation below.

You can retrieve your token by logging in via /api/login endpoint.

Admin endpoints

POST Category

requires authentication

Creates a new Category record.

Example request:
curl --request POST \
    "http://localhost/api/v1/admin/Categorys" \
    --header "Authorization: Bearer {YOUR_AUTH_KEY}" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"is_public\": true,
    \"name\": \"totam\",
    \"description\": \"et\",
    \"number_of_days\": 11
}"

const url = new URL(
    "http://localhost/api/v1/admin/Categorys"
);

const headers = {
    "Authorization": "Bearer {YOUR_AUTH_KEY}",
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "is_public": true,
    "name": "totam",
    "description": "et",
    "number_of_days": 11
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Example response (200):


{
    "data": {
        "id": "996a36ca-2693-4901-9c55-7136e68d81d5",
        "name": "My new Category 234",
        "slug": "my-new-Category-234",
        "description": "The second best journey ever!",
        "number_of_days": "4",
        "number_of_nights": 3
    }
}

Example response (422):


{
    "message": "The name has already been taken.",
    "errors": {
        "name": [
            "The name has already been taken."
        ]
    }
}
